    Βημα 1: Φτιαξτε repository απο το github.com

    Βημα 2: Φτιαξτε έναν φακελο στο PC

    Βήμα 3: Δωστε την εντολη git init μεσα στο φακελο

    Βήμα 4: Γράψτε κώδικα

    Βήμα 5: git add (ο,τι κανατε)

    Βήμα 6: git commit -m "βαζετε ενα μηνυμα για το τι αλλαξατε"

    Βήμα 7: Συνδεστε το Online repo με το local repo με την εντολη git remote add origin (και το url σας)

    Βήμα 9: Ανεβαστε τον κωδικα online με την εντολη git push origin master
