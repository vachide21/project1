  • Οδηγος Git

    Βημα 1: Ενας απο εμας εφτιαξε repository απο το github.com

    Βημα 2: Φτιαξαμε έναν φακελο στο PC

    Βήμα 3: Δωσαμε την εντολη git init μεσα στο φακελο για να γινει μερος του git

    Βήμα 4: Δωσαμε την εντολη touch index.html και δημιουργησαμε παρομοιως και τις αλλες σελιδες

    Βημα 5: Δωσαμε την εντολη git --global user.name και --global user.email για να εχουμε ενα ονομα μεσω του git

    Βημα 6: Δωσαμε την εντολη git remote add origin και το URL του repository μας(οχι αυτος που το εφτιαξε)

    Βημα 7: Αρχισαμε να γραφουμε την ασκηση

    Βήμα 8: git add για να προσθεσουμε οτι καναμε τοπικα

    Βήμα 9: git commit -m "βαζαμε ενα μηνυμα για το τι καναμε"

    Βήμα 10: Ανεβαζαμε τον κωδικα online με την εντολη git push origin master
